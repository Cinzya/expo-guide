# expo-guide
Web App for museums and expositions.

---

## Installation
```
$ npm install -g expo-cli
```

```
$ npm install
```

```
$ expo start
```

Starte iOS Emulation:
```
$ exp start --ios [options]
```

Starte Android Emulation:
```
$ exp start --android [options]
```
